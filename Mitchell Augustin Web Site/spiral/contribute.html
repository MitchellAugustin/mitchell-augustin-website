Bug Reports

I primarily built Spiral for myself, so as a result, I haven't done a ton of testing on every feature. If you find any bugs in Spiral, feel free to report them to Spiral's GitLab Issues Page, and I (or another maintainer) will get to work on fixing them.

Feature Requests

If you have an idea for a new feature, feel free to submit it to the GitLab Issues Page as well.

Code Contributions

Spiral is an open-source project, so anyone is welcome to contribute to its codebase, which can be found here on GitLab. If you would like to implement a feature or bug fix, pick one out from the issues page (or add your own) and comment that you are working on it in the corresponding thread.

Please make sure to add/comment on an issue before working on a feature to prevent any issues from arising after you finish making changes.

Once you've posted the issue to let me know you're working on a feature, fork the repository, make your changes, and submit a merge request. If everything checks out, I'll approve it and it will be added to the master branch.

Donations

I spent quite a bit of time and effort developing Spiral. If you like Spiral and want to show your support, any monetary donation would be greatly appreciated.

Other Feedback

If you have any other feedback on Spiral that doesn't fall into any of the above categories, feel free to send me an email at contact@mitchellaugustin.com and I'll try to address it as soon as I am able.
